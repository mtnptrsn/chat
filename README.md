# Chat

## Prerequisites
- node
- npm / yarn

## Installation
1. Clone the repository
2. Create a `.env` file in the root folder with the following properties:
  - REACT_APP_ENDPOINT
  - REACT_APP_TOKEN

Example:
```
REACT_APP_ENDPOINT="https://an-endpoint.ai"
REACT_APP_TOKEN="qwgqwgowgogewoigjoweig487937532"
```
3. Install packages by running `yarn` or `npm install`
3. Start app by running `yarn start` or `npm run start`
